# Student Management System

A full-stack CRUD application built using **Spring Boot** and **React** for managing student records.

## 🚀 Technologies Used

### Backend

- Java
- Spring Boot
- Spring Data JPA
- MySQL

### Frontend

- React
- Vite
- Axios
- CSS

---

## ✨ Features

- Add Student
- View Students
- Update Student Details
- Delete Student Records
- REST API Integration
- Responsive User Interface

---

## 📋 Student Entity

| Field         | Type    |
| ------------- | ------- |
| ID            | Integer |
| Name          | String  |
| Age           | Integer |
| Email         | String  |
| Mobile Number | String  |
| Marks         | Integer |
